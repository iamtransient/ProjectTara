# 🌸 Transient Mandala

> **A beautiful, interactive Mandala Generator built in a single HTML file.**
> No frameworks. No dependencies. Just HTML, CSS, and JavaScript.

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![HTML5](https://img.shields.io/badge/HTML5-Single%20File-orange)

![JavaScript](https://img.shields.io/badge/JavaScript-Vanilla-yellow)
![Responsive](https://img.shields.io/badge/Responsive-Yes-brightgreen)

---

## ✨ Features

* 🎨 Generate beautiful symmetrical mandalas
* 🌈 Custom color palettes
* 🎛 Adjustable number of petals and layers
* ⚡ Real-time rendering
* 🖱 Interactive controls
* 📱 Responsive design
* 💾 Export artwork as PNG
* 🌙 Dark mode interface
* 🚀 Runs completely offline
* 📄 Entire project contained in a **single `index.html` file**
